株価予測Webアプリケーション

さまざまな機械学習アルゴリズムを使用して株価を予測するWebアプリケーションです。ユーザーは銘柄コードを入力し、選択したモデルに基づいて株価の予測を行うことができます。

特徴

LSTM、XGBoost、ARIMA、GARCHなどの機械学習モデルを使用した株価予測

Q-learningによる強化学習を用いた予測

ユーザーが指定した日数先の株価の予測

予測結果をグラフで表示



銘柄コード入力例KDDIの場合:  9433.T


日数の入力例:              10



※予測株価の精度は保証できません
